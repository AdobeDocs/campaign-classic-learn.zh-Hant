# 產生和安裝 [!DNL GPG] 金鑰

[!DNL GPG (GnuPG)] 使用非對稱的單一金鑰組加密訊息。您可以使用 Adobe Campaign [!UICONTROL Control Panel]來安裝並產生 GPG 金鑰。

以下影片說明如何：

1. 瞭解如何在指定的 Campaign 執行個體產生和安裝&#x200B;**公開/私用金鑰組**，以&#x200B;**加密傳出資料**。
2. 如何在 Campaign 執行個體匯入並安裝&#x200B;**公開金鑰**，以&#x200B;**解密傳入的資料**。
