# 使用預先定義的篩選器設定疲勞管理

疲勞管理控制訊息傳送的頻率和數量，以避免收件者過度招攬。 如果您的促銷活動例項中沒有促銷活動最佳化模組，您可以設定預先定義的篩選條件，以依據收到的訊息數量來篩選目標人口。

本影片說明如何使用濾鏡在Adobe Campaign Classic實施疲勞管理。
