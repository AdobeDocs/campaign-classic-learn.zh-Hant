# 使用預先定義的篩選器設定疲勞管理

疲勞管理控制傳訊的頻率和數量，以避免過度向收件者發送請求。 如果您的市場活動實例中沒有市場活動優化模組，則可以配置一個預定義的篩選器，按接收的消息數篩選目標人口。

此視頻說明了如何使用過濾器在Adobe Campaign Classic實施疲勞管理。
