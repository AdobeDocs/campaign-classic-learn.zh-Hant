# 如何配置濃縮活動

擴充活動是允許使用者新增資料至工作表的工作流程物件。 使用擴充功能，您可以新增欄至該表格以儲存其他資料，或定義從該工作表到其他現有資料的新連結。

此視訊說明如何根據傳送記錄資訊來設定擴充活動。
