# 大資料管理Snowflake

本教學課程說明如何運用Adobe Campaign Classic的[!DNL Snowflake]連接器。
您將學習如何

* 連線至Snowflake
* 建立連結至Snowflake的架構
* 擴充架構並準備資料
* 建立工作流程和查詢
* 瞭解Snowflake
* 建立表格並讓行銷人員可存取
* 搜尋並報告資料
* 如何匯出清單中的資料並建立即時定位篩選
