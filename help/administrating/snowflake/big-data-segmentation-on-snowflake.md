# 雪花的大資料管理

本教學課程說明如何在Adobe Campaign [!DNL Snowflake] Classic中運用連接器。
您將學習如何

* 連接到雪花
* 建立連結至雪花的架構
* 擴充架構並準備資料
* 建立工作流程和查詢
* 瞭解雪花裡會發生什麼
* 建立表格並讓行銷人員可存取
* 搜尋並報告資料
* 如何匯出清單中的資料並建立即時定位篩選
